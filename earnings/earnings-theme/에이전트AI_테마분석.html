Executive Summary (v4)
  1. 에이전트AI = LLM이 multi-step reasoning + tool calls + 컨텍스트 유지로 자율 작업 실행.
  2. 수요 충격: 에이전트 1건 = chat 대비 토큰 20-30배 (코딩 1000배).
  3. 병목 = 추론 인프라 두 축: ① 반도체(HBM·레거시 메모리·광통신·CPU·추론 GPU·CoWoS·후공정) ② 전력(데이터센터 전력·UHV 변압기·SMR·가스터빈·액랭 + ★ v4 800VDC 전환).
  4. 병목 분류: ★ (A) 구조적 메가 병목 + (D) 동반 확대 hybrid. v4 800VDC = (D).
  5. 글로벌 트렌드 관통 + Moat 기업: SK하이닉스(HBM #1·DDR5 #2·eSSD #2), 한미반도체(TC본더 #1), 효성중공업(미국 765kV UHV #1), 두산에너빌리티 (SMR forging 글로벌 5사 중 1), 오이솔루션(1.6T·ITLA 국산화), ★ LS ELECTRIC (DC 분야 한국 선도, UL 1500V DC MCCB 최초) 등.

★ 신규 병목 #1

3-2-2. 레거시 메모리 (DDR5 server + eSSD) — KV cache offload

메커니즘: 토큰 폭증 → KV cache 폭발 → HBM 부족 → NVIDIA Dynamo·vLLM이 DDR5·eSSD로 offload. 레거시 메모리가 "추론 active 메모리"로 격상.

비유: HBM = 책상 위 책 (가장 빠름), DDR5 = 책장 책, eSSD = 도서관 책. 에이전트는 매번 100권 못 들어서 → DDR5/eSSD가 "확장 메모리".

DDR5 server 가격 추이 ($)

→ Samsung 32GB $149 → $239 (9월 +60%). DRAM YTD +50%, Q4 +30%

eSSD Top 5 분기 매출 ($B)

→ Q3 $6.54B → Q4 +50% QoQ. SK Group +75% QoQ로 30.2%

병목 카테고리: DDR5 = (A) 구조적 메가 병목 / eSSD = (A) 구조적 메가 병목 → (D) 동반 확대
