> 분기별 in-place 갱신. tier 분류는 분기 시작 시 BT가 직접 결정.

>
> **명칭 컨벤션**: 본 워치리스트의 T1-T3은 "섹터 T1·T2·T3"로 호칭한다 (이슈 워치리스트의 T1-T3과 구분).

## tier 운영 룰 요약

- **섹터 T1·T2 종목**: preview → review → in-depth 풀 사이클
- **섹터 T3 종목**: preview + review만 (in-depth는 시그널 감지 시)
- **Stage 2 깊이**:
  - 섹터 T1 풀 깊이 (가설·논쟁·통합)
  - 섹터 T2 가벼운 분석 (사이클·핵심 시그널)
  - 섹터 T3 시그널 감지 (T2 승격 후보 검출)

## tier 동적 조정

- 분기 중 강한 시그널 발생 시 BT가 승격·강등 결정

## 글로벌 피어 통합 운영 룰 (2026-05-03 신설)

(1) **분기 시작 시 발표 캘린더 정리** (BT 직접 또는 Claude 보조)
→ 한국 종목 vs 글로벌 피어 발표일 1주 단위 sequence 정리
→ 정렬 결과는 본 watchlist 종목 순서로 반영

(2) **선제적 발표 글로벌 피어 = 산업 사이클 cue**
→ ABB(4/16)·GEV(4/23)·SK하이닉스(4/23) 등 한국 3사보다 1주 빠른 발표 = 다음 한국 종목 분석 시 cross-reference 핵심 입력
→ 산출물: `2026-Q1_GEV_리뷰.md`, `2026-Q1_ABB_리뷰.md` 등 (한국 종목과 동일 표준 위치·메타데이터)

(3) **Stage 2 통합 분석**
→ Stage 2가 한국 + 글로벌 피어 review·in-depth 모두 자동 로드 (메타데이터 [섹터] 일치 필요)
→ 섹터 통합 사이클 분석 = 9개 종목 (전력 인프라) 같은 섹터로 처리

(4) **글로벌 피어 리뷰 깊이 (가벼움)**
→ 한국 종목 풀 사이클 (preview → review → in-depth) vs 글로벌 피어 review 위주 (in-depth는 산업 모멘텀 큰 변화 시만)
→ 글로벌 피어 review = 한국 종목 review 표준 7개 항목 중 핵심 4-5개 (실적 추이·컨센 비교·경영진 코멘터리·관전포인트) 압축 작성 권장

(5) **글로벌 피어 종목 표기·티커 컨벤션**
→ ABB(ABB), GE Vernova(GEV), Eaton(ETN), Schneider Electric(SU), Hitachi Energy(HE), Siemens Energy(ENR)
