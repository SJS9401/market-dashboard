> 안내: 표준 위치(`earnings-preview/`)에서 동일 (분기, 종목)의 프리뷰 .md를 자동 검색한 결과 **프리뷰 자료 없음** — 본 분기 단독 분석으로 진행. 항목 4-1·7-1(프리뷰 비교) 자동 생략. 그 외 모든 항목은 정상 작동.

## Executive Summary

→ **역대급 트리플 비트**: 매출 $23.86B (가이던스 $18.7B 대비 **+27%**, 컨센 $20.07B 대비 +19%) / Non-GAAP EPS $12.20 (가이던스 $8.42 대비 **+45%**, 컨센 $9.31 대비 +31%) / Non-GAAP GPM **74.9%** (가이던스 68% 대비 +7pp). 4분기 연속 매출 신기록 + 분기 매출 sequential 증가폭 $10.2B로 사상 최대.
→ **DRAM ASP가 단일 분기 +mid-60%** 점프 — 비트는 mid-single digits만 증가했음에도 가격이 모든 것을 설명. NAND ASP도 +high-70% 폭등. 이는 메모리 사이클 전체의 구조 전환 시그널.
→ **FQ3 가이던스가 더 충격**: 매출 $33.5B (±$750M), GPM **~81%**, EPS $19.15. CEO Mehrotra "FQ3 단일 분기 매출 가이던스가 FY24까지의 **연간 매출을 초과**." 컨센 $24.29B 대비 가이던스 **+38%** 상회.
→ **공급 측 락인 확정**: 첫 5년 SCA 체결, HBM4 12H 본격 출하 (NVIDIA Vera Rubin 향), HBM4 16H(48GB) 샘플링, FY26 CapEx **>$25B로 상향**(전 가이던스 +$5B), FY27 CapEx 추가 +$10B 이상.
→ **시장 반응의 역설**: 발표 직후(3/19) 주가 -4% sell-the-news → 3/30 -30% 폭락(TurboQuant 공포) → 4월 회복 → 5/1 종가 **$540.15** (1개월 +61%, 1년 +522%). 펀더멘털과 주가가 4월 한 달 만에 재정렬됨.

→ FY27 매출 $155B, EPS $80+ 시나리오 가능
→ 메모리 사상 최강의 사이클 상태 유지

(2) 시장 반응 해석
→ 발표 직후 -4%, 3/30 -30% 폭락은 "TurboQuant 충격" + "CapEx 상향 우려" 결합
→ 그러나 1개월 만에 회복 + 사상 최고가 재돌파 ($540, 5/1)
→ → "narrative shock"보다 **fundamentals**가 결국 이김
→ 평균 목표주가 +50% 상향 = 시장이 cyclical → secular로 인식 전환

(3) 사이클 핵심 시그널 (가장 중요한 선행지표)

(3-1) **DRAM Spot vs Contract Price 갭** (Trendforce, DRAMeXchange)
→ 현재: contract 가격이 spot price를 큰 폭 상회 (이례적)
→ 정상: spot이 약간 더 높음 (즉시 출하 프리미엄)
→ 시그널: contract > spot 지속 = **공급 부족 심각** + 마이크론 가격 협상력 우위
→ 전환 시그널: spot이 contract를 추월하는 시점 = 사이클 정점 임박

(3-2) **HBM 트레이드 비율** (Trendforce)
→ HBM 1bit 생산에 일반 DRAM 대비 약 3배 wafer 소비
→ HBM mix 상승 → DRAM 일반 공급 추가 감소 → 가격 추가 상승
→ 시그널: HBM 비중이 산업 DRAM의 30%+ 도달 시 가격 모멘텀 추가

(3-3) **Hyperscaler CapEx YoY 변화율**
→ 현재 +77% YoY (2026E)
→ 2027 가이던스 발표 (2026 Q4 결산) 시점 = 사이클 변곡점 가능성
→ 시그널: hyperscaler CapEx YoY가 +30% 미만으로 둔화 시 메모리 사이클 후기 진입

(3-4) **마이크론·SK하이닉스 inventory days**
→ 마이크론 DRAM DOI **<120일** (이미 매우 타이트)
→ DOI가 100일 미만으로 추가 하락 시 "재고 거의 없음" → 즉시 출하 비중 100%
→ DOI가 130일 이상으로 상승 시 사이클 전환 시그널

(3-5) **CapEx 발표 추이 — 3사 합산**
→ 마이크론 FY26 >$25B + SK하이닉스 + 삼성 메모리 = 3사 합산 약 $80B+
→ 시그널: 3사 합산 CapEx가 $100B+ 도달 시 2028~2029 공급 과잉 우려

(4) 사용자 별도 체크 항목 (확장 가능)
→ 마이크론 HBM4E 16H 양산 시점 (CY2027 예정 vs 실제)
→ Tongluo fab의 의미 있는 출하 시점 (FY28 예정 vs 가속 가능성)
→ 마이크론 vs SK하이닉스 vs 삼성 분기 매출 격차 정량화 (SK하이닉스 1Q 2026 매출 52.6조원 = 약 $35.5B vs 마이크론 Q2 FY26 $23.86B)
