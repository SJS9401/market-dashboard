Alphabet — 2026 Q2 팔로업 리포트

⚠️ 첫 작성 안내: 본 종목 4Q25 리뷰 자료가 워크스페이스에 부재 → (2) 지난 분기 핵심 팔로업 사항 섹션 자동 생략. v6 룰 적용 첫 사이클이므로 본 리포트가 baseline. 차기 분기(3Q26)부터는 (2) 섹션이 정상 작동.

(1) 1Q26 실적 — Beat/Miss/In-line 동인 분해

All-around Beat — M7 1분기 발표 종목 중 최강의 펀더멘털 분기. 매출 $109.9B (+22% YoY, +19% constant currency)는 컨센 +1.3% Beat이자 11분기 연속 두 자릿수 성장 기록. GAAP OPM 36.1% (+220bps YoY)는 cons 34.5% 대비 +160bps Beat. 매출 +22% > 총비용 +18% = operating leverage 명확 입증. Search 매출 $60.4B (+19% YoY) + YouTube ads $9.88B (+11%) + Subs $12.38B (+19%) 모두 가속, Network -4%는 4분기 연속 마이너스로 구조적 약세이나 매출 비중 6.3%로 임팩트 제한적.

진짜 헤드라인은 Cloud 트리플. 매출 $20.0B (+63% YoY) 첫 $20B 돌파, OPM 17.8% → 32.9% (+1,510bps YoY 트리플), 영업이익 $6.6B (+203% 트리플). 백로그 $462B (QoQ ~2배) — TPU 하드웨어 외부 판매 + multi-billion deals 포함. CFO Anat: "expect to recognize just over 50% of the backlog as revenue over the next 24 months". Sundar verbatim "*we are compute constrained in the near term. Cloud revenue would have been higher if we were able to meet the demand*" — 즉 공급 제약이 천장**, 백로그 폭증의 본질.

EPS Mega Beat의 본질은 일회성. Diluted EPS $5.11 (+82% YoY)는 cons $2.85 대비 +79% Beat이지만, 이 중 $2.35는 비마케터블 equity securities 평가차익 ($36.9B 평가, 세후 $28.7B — 주로 Anthropic 지분 추정). 영업 정상화 EPS는 약 $2.76로 cons와 유사. CapEx 가이던스 5번째 상향 ($175-185B → $180-190B), 2027 "significantly increase" 명시 + 자사주 매입 Q1 $0 (Q1 25 $15B 대비 완전 중단) + senior unsecured notes $31.1B 신규 발행 = 자본배치 우선순위 = AI CapEx ≫ Buyback 명확 시그널.

(2) 지난 분기(4Q25) 핵심 팔로업 사항

⚠️ 첫 작성 — 4Q25 리뷰 자료 없음. 본 분기(1Q26)부터 follow-up 사이클 시작. 차기 분기(2Q26 발표 후 3Q26 팔로업) 작성 시 본 리포트의 (4) 2Q26 핵심 팔로업 사항이 (2)로 이월되어 사후 검증 대상이 됨.

(3) 1Q26 핵심 변화

산업·사이클 측면: Cloud 가속 사이클 진입 확정 — Q3 25 +35% → Q4 25 +47% → Q1 26 +63% (3분기 연속 성장률 가속). 백로그 $462B는 2~3년 매출 가시성 확보 (50%+ 24개월 인식 가이드). 동시에 Sundar "compute constrained" 시인 = AI 인프라 사이클이 정점이 아닌 가속 단계. Search BU도 "queries at all time high" + AI Overviews/AI Mode 광고 monetization 확장으로 2차 성장 사이클 진입. Other Bets는 Verily 디컨솔 + GFiber 매각 발표로 포트폴리오 슬림화 방향 전환.

사업부 mix 변화: Cloud 매출 비중이 18.2% (Q1 25 13.6%) → +460bps로 사상 최고 갱신. Cloud OPM 32.9%는 "AI 매출은 마진 낮다"는 시장 선입견을 IR이 직접 반박. Wiz 인수 ($29.5B) + Intersect ($5.9B) Q1 26 within 흡수 — Wiz는 Cloud 사이버보안 BU로 통합되며 "low single digit pp 헤드윈드" Q2 이후 발생 예정. Services 비중 81.6%로 다소 감소했으나 Search/YouTube/Subs 모두 두 자릿수 성장. 350M paid subscriptions 도달 (Sundar) — YouTube + Google One AI plans 결합 ARR 모델.

Narrative·Multiple 측면: AI 풀스택 차별화 narrative가 IR Q1 26에서 다중 데이터 포인트로 입증 — Gemini Direct API 16B+ tokens/min (+60% QoQ), 330개 고객 >1T tokens 처리, Gemini Enterprise paid MAU +40% QoQ, "products built on gen AI models 매출 +800% YoY". TPU 8 시리즈 출시 (TPU 8t 학습 / TPU 8i 추론 80% 성능/달러 개선) + 외부 직접 판매 채널 신설 = 하드웨어 매출 다각화 시작. 컨센 등급 강화 (PT 평균 $200 → $215, +7.5%, Strong Buy 비중 +4명) — Wedbush·Bernstein·MoffettNathanson 등급 상향. AI 사이클 M7 best-positioned 종목으로 narrative 견고화. Antitrust risk는 항소심 진행 중 (Search 분리 명령 가능성)으로 multiple 캡 잠재.

(4) 2Q26 핵심 팔로업 사항 (4~5 키워드)

키워드 1: Cloud 백로그 모멘텀 + 매출 인식 페이스

Q1 26 백로그 $462B (QoQ ~2배)에서 Q2 모멘텀이 가장 중요한 시그널. Q2 26 백로그 $500B+ 도달 시 → 가속 지속, 2027 CapEx 추가 상향 정당화. $400~470B 권역 정체 시 → "compute constrained 해소" 시그널. CFO 가이드 "50%+ 24개월 인식"은 분기 평균 ~$48B Cloud 매출 잠재 — Q2 Cloud 매출이 $21.5B+ 도달 시 백로그 인식 페이스 정상.
→ 모니터링 채널: Q2 26 Earnings Release Cloud RPO disclosure (7월 말 예정), Cloud Next 2026 (10월) 사전 시그널
→ 뉴스 키워드: "Google Cloud backlog", "GCP RPO", "Google Cloud growth", "Sundar compute capacity"

키워드 2: Cloud OPM Trajectory (Wiz 헤드윈드 검증)

CFO Anat 직접 가이드 "low single digit percentage point headwind to Cloud's operating margin for the remainder of 2026 related to the [Wiz] acquisition" → Q2 Cloud OPM 30~31% 권역으로 약간 후퇴 예상. Q2 Cloud OPM 28% 이하로 떨어지면 통합 비용 추가 우려, 33%+ 도달 시 Wiz 시너지 조기 가시화 (Strong Buy 시그널). 이 분기가 "Cloud OPM 32.9%가 sustainable인지" 1차 검증.
→ 모니터링 채널: Q2 26 Earnings Release Cloud segment OPM, Wiz 매출 separate disclosure 여부
→ 뉴스 키워드: "Wiz integration Google Cloud", "Cloud margin Q2", "Google Cloud OPM"

키워드 3: CapEx Q2 페이스 (FY26 $180-190B 가이드 검증)

FY26 가이드 $180-190B 달성을 위해 Q2~Q4 평균 $48~52B 페이스 필요 — Q1 $35.7B는 baseline 미달. Q2 CapEx $45~48B → 가이드 정상, $50B+ → 추가 상향 가능성, $40B 미달 → 가이드 하단 위협 + 2027 가속 정당화 어려움. CFO "60% 서버 / 40% 데이터센터·네트워킹" 분배 비율 변화도 추적. 2027 CapEx "significantly increase" 정량화 시점은 Q3 26 컨콜 가능성.
→ 모니터링 채널: Q2 26 Earnings Release CapEx 항목 + CFO 가이드 멘트, Hyperscaler CapEx 비교 (MSFT/AMZN/META Q2 발표)
→ 뉴스 키워드: "Google CapEx Q2", "GOOGL data center construction", "2027 CapEx guidance"

키워드 4: Search AI 광고 coverage 확장 (20% → 25%+)

CBO Schindler Q1 답변 "I think there is upside in that coverage number" — 현재 20% coverage가 25%+로 확장 시 Search 매출 +α. AI Overviews + AI Mode가 commercial queries에서도 광고 노출 확대. Q2 reported 광고 coverage 변화 + Etsy/Hilton 케이스 같은 advertiser ROI 데이터 추적.
→ 모니터링 채널: Q2 26 Earnings Slides Search & Other 분해, Google Marketing Live 발표 (5월 21일 예정)
→ 뉴스 키워드: "Google Search ads coverage", "AI Overviews monetization", "AI Max Performance Max", "Universal Commerce Protocol"

키워드 5: TPU 외부 판매 첫 매출 인식 + Gemini 4 출시

CFO Anat "small percent of revenues from these agreements later this year, with the vast majority of revenues to be realized in 2027" — Q2 26 첫 의미 있는 매출 인식 ($1B+) 도달 시 2027 ramp 정상. 동시에 Gemini 4 출시 가능성**을 5월 19일 I/O 후속 모멘텀으로 추적. TPU 8t/8i 양산 페이스 + Vera Rubin GPU 도입 페이스도 추적.
→ 모니터링 채널: Google I/O 2026 (5월 19일) 발표, Q2 Earnings Release TPU 매출 disclosure, Anthropic 평가 변동
→ 뉴스 키워드: "Google I/O 2026", "Gemini 4 release", "TPU 8 customers", "Google Anthropic stake"

키워드 6: Antitrust 항소심 + Anthropic 평가 변동성

DOJ Search Antitrust 항소심 결정 시점 (Q3-Q4 26 가능)으로 단기 multiple 캡. Search 분리 명령 시 매출 ($240B+/year) 직접 임팩트. 동시에 비마케터블 equity gain $36.9B (Anthropic 추정)의 Q2 평가 변동이 GAAP EPS 변동성 확대 — Q2 EPS는 일회성 제거 정상화 영업 EPS 추세 추적이 핵심.
→ 모니터링 채널: DOJ vs Google Antitrust 법원 일정, Anthropic 펀딩 라운드 평가 변동
→ 뉴스 키워드: "Google antitrust appeal", "Search remedy ruling", "Anthropic valuation Series", "Google equity gains"
