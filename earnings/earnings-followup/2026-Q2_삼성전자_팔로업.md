## ① 1Q26 실적 — 비트/미스/보합 요인

(1) 사상 최대 분기 + 한국 기업 최초 OP 50조 돌파
→ 매출 **133.9조원** (잠정 4/7 +33% 서프라이즈, 확정 4/30 동일)
→ 영업이익 **57.2조원** (컨센 약 40조 대비 **+43%** / +185% QoQ / **+756% YoY**)
→ OPM **42.8%** (전 분기 21.4% → +21.4pp 단일 분기 점프)
→ EPS 7,123원, ROE **41%** (1Q25 8% → 5배 점프)

(2) 비트 핵심 동인 — DS 메모리가 사실상 모든 것
→ DS 단독 OP **53.7조 = 전사의 94%** (메모리 OP 약 50조 추정)
→ DRAM ASP **+90~93% QoQ** — 마이크론(+mid-60%)·SK하이닉스(+63~65%) **능가** (4Q25 base 낮음 + HBM4 양산 효과)
→ NAND ASP +89% QoQ + Bit +9% (SK -13% 대비 우위) — 출하 모멘텀 견조
→ HBM4 NVIDIA Vera Rubin 양산 시작 + SOCAMM2 동시 양산
→ MX/NW (S26 출시 효과 +33% QoQ 매출), Foundry (선단공정 가동률 최고)

(3) 미스/보합
→ 미스 항목 부재 (전사 OP 모든 컨센 비트)
→ 보합: MX 7% OPM (역사 정상 10~12%), VD/DA 1% OPM (메모리 비용 부담), Harman 6% OPM
→ → 비반도체 사업부 모두 OPM 약세 — 메모리 가격 폭등 → 자체 부품 원가 부담의 양면성

## ③ 1Q26 핵심 변화 (3개 카테고리)

(1) 산업/사이클
→ **메모리 산업 전체 슈퍼사이클 입증** — 마이크론(3/18) → 삼성 잠정(4/7) → SK(4/23) → 삼성 확정(4/30) 6주간 4단계 시퀀스로 record 갱신
→ **삼성이 글로벌 메모리 매출 1위 확정** — DS 81.7조 ($57B) > SK 52.6조 ($37B) > 마이크론 $23.9B
→ DRAM/NAND ASP 동시 +90% 폭등 — 산업 광범위 사이클
→ **LTA 다년 계약 추진 또는 완료** (4/30 컨콜 명시) — 마이크론 SCA·SK LTA와 동일 흐름, spot 가격 시대 종료
→ "**2027년이 2026년보다 더 타이트할 것**" — 회사 가이드 (한화 인용)

(2) 사업부 mix
→ **DS 매출 비중 32% → 61% 점프** (1Q25 → 1Q26 단일 분기) — DS가 사실상 회사 자체로 전환
→ **HBM4 12H NVIDIA Vera Rubin 양산 시작** — SOCAMM2 동시 양산
→ **HBM4E 첫 샘플 2Q26 공급 예정** (회사 가이드)
→ Foundry 선단공정 가동률 최고 수준 + **1.4nm 개발 순항, 2nm 대형 수주 추진**
→ MX/VD/Harman: 메모리 비용 부담으로 OPM 약세 — DS 호황의 양면성

(3) Narrative / Multiple
→ **한국 8개 증권사 평균 PT +41% 상향** (4월 초 230,000 → 5/4 322,000원)
→ Strong Buy 만장일치 (KB·유진 360,000 / IBK 350,000 / 하나·한화 330,000)
→ 유진 Strong Buy **신규 진입** — "단기 부담을 이겨낼 구조적 변화"
→ KB "메모리 = TSMC식 파운드리화" + 휴머노이드 로봇 협업 (레인보우로보틱스, 2H26 공개)
→ ★ **Citi 5/3 PT cut (32만→30만, -6.3%)** — 노조 우려로 단일 보수 시각 (한국 평균 32.2만 대비 -6.8%)
→ **순현금 119조 + 자기주식 매입 7.6조 (4Q25 0.04조 대비 폭증)** — 100조 trigger 후 환원 가속

## ④ 2Q26 핵심 팔로업 사항

(1) **★★ 노조 파업 5/21~6/7 D-day** (1순위 — 단기 변동성 결정 핵심)
→ 4/30 노조 majority status 획득 → 5/21 풀 파업 위협 (요구 미수용 시)
→ 노조 요구: OP 15% 보너스 = 약 45조원 (Citi 추정 충당금 30조 비용 위험)
→ 시나리오 A(5/20 이전 타결, 40% 확률) / B(단기 1~2주 파업, 35%) / C(장기 3주+, 20%) / D(총파업, 5%)
→ **모니터링**: 전국삼성노조연대 SNS, 삼성전자 IR 보도자료, Citi 추가 PT 변경, JPM/MS/Goldman 동조 여부, 5/21 D-day 1주 전 한국 8개 증권사 추가 코멘트
→ **뉴스 키워드**: "삼성전자 노조 파업" / "Samsung union strike" / "성과급 충당금" / "보너스 협상"

(2) **2Q26 OP 87조 도달 여부 + DS OP 80조**
→ 컨센 87조 (5/4 8개 평균, 범위 82.8~91.6조)
→ 90조+ 달성 → secular 강화, 추가 +10% 가능 / 80조 미달 → 노조 영향 가시화
→ **2Q26 DRAM ASP +33~42% QoQ (애널 평균), NAND +50~60% QoQ** — 1Q26 +90% 후 자연스러운 둔화이나 절대 수준은 사상 최고 유지
→ HBM4 출하량 vs 가이드 (1Q26 추정 1.5만개 → 2Q26 2.5만개+)
→ **모니터링**: 잠정실적 7/8 발표일 + IR 컨콜 톤, FnGuide 컨센 직후 갱신

(3) **HBM4E 2Q26 첫 샘플 공급 진척 + HBM 점유율 회복**
→ 회사 명시: "기술 리더십을 공고히 하기 위해 HBM4E 첫 샘플 공급 예정"
→ HBM 점유율 23% → 25%+ 도달 가능성 (SK 70% 점유 대비 추격 단계)
→ HBM4E 1c 코어 다이 적용 (마이크론·SK와 동일 노드)
→ **모니터링**: TrendForce HBM 분기 보고서, NVDA 어닝 (5/20) HBM 공급사 코멘트, Bloomberg "HBM4E sampling Samsung"

(4) **Foundry 1.4nm 개발 + 2nm 대형 수주**
→ "1.4나노 개발 순항, 2나노 대형 수주 확대 추진" (4/30 컨콜)
→ HBM4 B-die 등 공급 증가로 Foundry 실적 개선 목표
→ 2H27 파운드리 흑자 전환 가능성 (유진 인용)
→ Apple/Qualcomm/Tesla 중 어느 고객사가 2nm 대형 수주로 발표될지
→ **모니터링**: 삼성전자 Foundry Forum (Foundry 수주 발표), Bloomberg "Samsung 2nm customer"

(5) **자사주 매입소각 추가 발표 + 순현금 100조 후 환원 가속**
→ 1Q26 자기주식 매입 7.61조원 (4Q25 0.04조 대비 폭증) — 노조 우회 대응 + 환원 가속 시그널
→ 2Q26 추가 매입 규모 발표 가능성 (3~5조 추정)
→ 1Q26말 순현금 119.24조 → 2Q말 추정 140조 → 100조 trigger 후 추가 환원 강화 가능
→ **모니터링**: 삼성전자 IR 보도자료, 분기 자기주식 매입 공시 (DART)
