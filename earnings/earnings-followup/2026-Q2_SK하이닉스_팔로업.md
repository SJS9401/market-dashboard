## ① 1Q26 실적 — 비트/미스/보합 요인

(1) 사상 최대 분기 + OPM 71.5% 한국 제조업 사상 초유
→ 매출 **52.58조원** (FnGuide 컨센 51.9조 대비 **+1.3%** / 직전 4Q25 32.83조 대비 **+60.2%** QoQ / 전년 1Q25 17.64조 대비 **+198.1%** YoY)
→ 영업이익 **37.61조원** (컨센 36.4조 대비 **+3.3%** / +96.2% QoQ / **+405.5%** YoY)
→ OPM **71.5%** (컨센 70% 대비 +1.5pp) — 마이크론 GAAP OPM 67.6% 능가, **글로벌 메모리 OPM 1위 확정**
→ 순이익 40.35조원 (컨센 30.8조 대비 **+31%** 큰 폭 비트 — 키옥시아 평가이익 + 환율 일회성)
→ GPM **79.3%** (1Q25 16% → +63pp 1년 만에 점프)

(2) 비트 핵심 동인 — 가격이 거의 모든 것
→ DRAM ASP **+63~65% QoQ** (Bit -0~+0%) — 가격이 거의 모든 매출 성장 견인
→ NAND ASP **+73~74% QoQ** (Bit -11~-13%) — 더 큰 폭등
→ DRAM OPM 78~80% / NAND OPM 49~52% — 양 사업부 모두 사상 최고
→ 메리츠 추정 **커머디티 DRAM OPM 86%** = 글로벌 메모리 사상 단일 제품군 최고 마진

(3) 보합/미스
→ 매출/OP 모두 컨센 부합 수준 비트 (1~3%) — 프리뷰의 "40조원+ 돌파" / "OPM 75%+" 강한 시나리오는 빗나감
→ Beat 폭 4Q25 +16% → 1Q26 +3.3%로 **큰 폭 축소** (컨센이 발표 직전 +82% 상향으로 이미 따라잡음)
→ NAND Bit -13% (큰 폭 감소) — 솔리다임 대련 Phase 2 ramp 지연 + 단품 판매 축소 + 4Q25 기저 결합
→ 미스 항목 부재 (모든 KPI 컨센 동시 비트 또는 in-line)

## ③ 1Q26 핵심 변화 (3개 카테고리)

(1) 산업/사이클
→ **메모리 산업 전체 슈퍼사이클 입증** — 마이크론(3/18) → 삼성 DS OPM 38% (4/7) → SK하이닉스 OPM 71.5% (4/23) 6주간 3사 동시 record
→ DRAM/NAND ASP QoQ — 마이크론(+mid-60%/+high-70%)과 SK하이닉스(+63~65%/+73~74%) **거의 동일 패턴** = 산업 전체 공급 부족
→ **HBM3E + HBM4 multi-year sold out** — 가격 가시성 본격 확보
→ **개선된 LTA** 적극 검토 (회사 컨콜 직접 확인) — 마이크론 첫 5년 SCA와 동일 흐름, spot 가격 시대 종료 시그널
→ 회사 코멘트: "현재 가격 상승은 일시적 수급 불균형이 아니라 시장의 구조적 변화" / "수급 부족 사이클 과거 대비 장기화 가능성 높음"

(2) 사업부 mix
→ **HBM4 12H 본격 양산** (NVIDIA Vera Rubin 향) — 마이크론과 동시 양산
→ **NAND OPM 1Q25 1% → 1Q26 50%대 점프** — 솔리다임 인수 5년 만에 본격 흑자 전환
→ **MCBU·CDBU 모두 OPM 70%+** — DRAM 일반 + 서버 모두 폭발
→ DRAM 비중 78% / NAND 21% (마이크론 79%/21%과 거의 동일 구조)
→ **321단 NAND 전환 본격화** — 올해 내 국내 생산량 50%+ 전환 (176단 → 321단 직접 점프)

(3) Narrative / Multiple
→ **8개 한국 증권사 평균 TP +25% 상향** (1,520,000 → 1,815,000원), 만장일치 매수
→ **메리츠/KB 200만원**, 신한 190만, NH·삼성 180만, DB 175만, 한화 163만, DS 160만 — Strong Buy 5건 + Buy 3건 + 매도 0건
→ KB 시각 전환: **"메모리 = TSMC식 파운드리화로 진화"** — 가장 강한 secular reframe
→ 메리츠 시각: "ADR 재평가 기회" + "마이크론 P/B 5.31 vs 하이닉스 2.66 = 절대 저평가"
→ 한화 시각: "이제부터는 폭보다 지속성" — cyclical → secular 시각 전환
→ **순현금 100조 트리거** — 1Q26 +6.5조 → 2026말 +153조 추정 → 100조 도달 시점 = **2026 연말~2027 H1**

## ④ 2Q26 핵심 팔로업 사항

(1) **★ OPM 76% 돌파 여부 (1순위)**
→ 1Q26 71.5% → 2Q26 컨센 76% (+4.5pp)
→ 76%+ 달성 → secular narrative 강화, 멀티플 영구 재평가
→ 76% 미달 → cyclical peak 우려 즉시 부각, sell-the-news 가능성
→ GPM 80% 돌파 여부 (신한 추정) — 2Q 마이크론 가이 GPM 81%와 정면 비교
→ **모니터링**: 잠정실적 2026-07 말 발표일 + 컨콜 Q&A "OPM sustainability" 톤, 한국 8개 증권사 즉시 PT 갱신

(2) **NAND Bit Growth 회복**
→ 1Q26 -13% → 2Q26 컨센 +13~15% QoQ
→ 솔리다임 대련 Phase 2 ramp + 단품 판매 회복 핵심
→ 회복이 컨센 +15% 미달 시 → 솔리다임 지연이 더 큰 영향이라는 시그널
→ Seagate/WDC/Sandisk 5월 초 발표 결과 → NAND 산업 컨센 추가 상향 가능성
→ **모니터링**: 한국 5월 반도체 수출 데이터 (5/11 발표), TrendForce NAND 가격 보고서

(3) **순현금 100조 진척률 + 자사주 매입소각 발표**
→ 1Q26 순현금 +6.5조 → 2Q26 +25~30조 추정 → 100조까지 +70조 남음
→ 곽 사장 컨콜 "이익 창출력을 고려하면 순현금 100조 + 주주환원 확대 동시 가능" 명시
→ "자사주 매입 소각 등 연내 마련해서 시장과 소통" — **2026 H2 자사주 환원 강화 발표 가능성 매우 높음**
→ **모니터링**: SK하이닉스 IR 보도자료, 곽 사장 추가 발언, 분기 FCF 산출 추적

(4) **★ ADR 6월 SEC 결정 + 미국 상장**
→ "ADR 미국 시장 상장 6월 이내 결정" (4-23 컨콜)
→ 신주 발행 방식 전환 (자사주 14조 소각으로 부족) → EPS 희석 우려 vs 미주 자본 유입 이점
→ 마이크론 P/B 5.31 vs 하이닉스 2.66 = +99% 멀티플 갭 축소 가능성
→ **모니터링**: SEC EDGAR 등록신청서 공시, SK하이닉스 IR 6월 보도자료, 외국인 지분율 변화 (현재 53.1%)

(5) HBM4 양산 진척 + HBM4E 샘플 일정
→ HBM4 12H NVIDIA Vera Rubin 본격 출하 진행 중 — 분기 매출 별도 공시 가능성
→ HBM4E: 하반기 샘플 공급, 2027 양산, 1c 코어 다이 적용
→ HBM 점유율 70% 유지 여부 (마이크론·삼성 추격 시그널)
→ **모니터링**: TrendForce HBM 분기 보고서, NVDA 어닝 (5/20) HBM 공급사 코멘트
