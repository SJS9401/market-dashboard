Amazon — 2026 Q2 팔로업 요약

⚠️ 첫 작성 안내: 본 종목 4Q25 리뷰 자료가 워크스페이스에 부재 → (2) 지난 분기 핵심 팔로업 사항 섹션 자동 생략. v6 룰 적용 첫 사이클이므로 본 리포트가 baseline. 차기 분기(3Q26)부터는 (2) 섹션이 정상 작동.

(1) 1Q26 실적 — Beat/Miss/In-line 동인 분해

All-around Mega Beat — Q1 26 OPM 13.1% "highest operating margin ever" (CFO Olsavsky verbatim). 매출 $181.5B (+17% YoY, +15% ex-FX) = 컨센 $178B 대비 +$3.5B Beat. Operating income $23.85B (+30% YoY) = Q4 25 컨콜 가이드 상단 $18B 대비 +$5.85B Mega Beat. 모든 세그먼트 Beat: NA $104.1B (+12% YoY, OPM 7.9%), International $39.8B (+19% YoY, OPM 3.6%), AWS $37.6B (+28% YoY, OPM 37.7%). M7 4/29 발표 4종(GOOGL/MSFT/AMZN/META) 모두 Beat 일관 톤.

진짜 헤드라인은 AWS + 칩 백로그. AWS +28% YoY = Jassy verbatim "fastest growth rate in 15 quarters" + $2B QoQ revenue increase = "largest Q4 to Q1 AWS revenue increase ever". AWS 연환산 run rate $150B 도달, AWS AI 매출 run rate $15B+ (3년 만에 AWS 초기 대비 260배). AWS 백로그 $364B (Anthropic $100B+ deal 별도 → 사실상 $464B+) + Trainium revenue commitments >$225B. Jassy: "Trainium2 largely sold out, Trainium3 nearly fully subscribed, Trainium4 already been reserved". 칩 비즈니스 $20B annual run rate (triple-digit YoY) = standalone $50B run rate, 데이터센터 칩 글로벌 톱 3 진입.

Net Income $30.3B Mega Beat의 본질은 일회성: $16.8B pre-tax는 Anthropic 평가차익. EPS $2.78 중 약 $1.22는 일회성 → 정상화 영업 EPS ~$1.56. TTM CapEx $147.3B (+67% YoY) = M7 hyperscaler 빅3 (AMZN/MSFT $144B/GOOGL $185B) 동등 수준. TTM FCF $1.2B (-95% YoY) = $25.9B → $1.2B 거의 zero, AI 인프라 사이클 첫 자본 시험. Q1 단독 CapEx $44.2B = M7 분기 CapEx 신기록 추정.

(2) 지난 분기(4Q25) 핵심 팔로업 사항

⚠️ 첫 작성 — 4Q25 리뷰 자료 없음. 본 분기(1Q26)부터 follow-up 사이클 시작. 차기 분기(2Q26 발표 후 3Q26 팔로업) 작성 시 본 리포트의 (4) 2Q26 핵심 팔로업 사항이 (2)로 이월되어 사후 검증 대상이 됨.

(3) 1Q26 핵심 변화

산업·사이클 측면: AI hyperscaler arms race 정량 확정 — TTM CapEx 빅3 동등 수준 (AMZN $147B / MSFT FY26E $144B / GOOGL $147B 누계). AWS +28% 5분기 연속 가속 + 백로그 $464B+ (Anthropic 포함) = 2~3년 매출 가시성. M7 4/29 발표 4종 모두 클라우드 BU 가속 (GOOGL Cloud +63%, MSFT Azure +40%, AWS +28%) — AI 인프라 사이클 정점 미도래의 sector-wide 시그널. Stores 사이클은 unit growth +15% (covid 이래 최대) + 마진 회복 sustained = AMZN consumer BU의 long-run cycle 진입. 광고 BU TTM $70B+ = Google YouTube ads 수준 도달 = 광고 시장 톱 3 진입.

사업부 mix 변화: AWS 매출 비중 19%→21% (+2pp). NA + International OPM +60~160bps YoY 동시 확장. 광고 +22% ex-FX 4분기 연속 = 매출 비중 9.5% 도달. Trainium 칩 비즈니스가 standalone $50B run rate = 새로운 BU의 부상 (Jassy: "one of the top three data center chip businesses in the world"). Bedrock 125,000+ 고객, 80% Fortune 100 사용. OpenAI 2GW + Anthropic 5GW + Meta tens of millions Graviton cores = 3대 AI lab + 빅테크 동맹 다중화. AMZN AI 인프라가 자체 chip + Bedrock + Trainium + Graviton 풀스택으로 진화.

Narrative·Multiple 측면: Jassy "we are unusually well positioned for this AI inflection" + "at scale, Trainium will save tens of billions of CapEx + 수백 bps OPM advantage" = AWS의 multi-year operating leverage narrative 강화. Operating margin record 13.1%는 Stores BU 정상화 + AWS 회복 + 광고 가속 + Trainium 효율의 합산 = 일회성보다 multi-segment 동시 회복 시그널. 컨센 Strong Buy 강화 (50명 PT 평균 $245→$258 +5.3%, Wedbush·BofA·MoffettNathanson Strong Buy 등급 진입, Barclays·UBS Hold→Buy 등급 상향). M7 4/29 4종 모두 톤 강화 일관 — TSLA 4/22 양극화와 명확한 대비.

(4) 2Q26 핵심 팔로업 사항 (5~6 키워드)

키워드 1: AWS Q2 +28%+ sustainability + 백로그 페이스 (최우선)

Q2 AWS YoY +28% 도달 시 → 6분기 연속 가속 + Anthropic $100B+ deal 매출 인식 시작. Q2 +30%+ 도달 시 → Bull case (Strong Buy 시그널), +25% 이하 → 가속 사이클 둔화 시그널. 백로그 trajectory: $364B → Q2 $400B+ (Anthropic 명시 포함) 도달 여부. AMZN은 GOOGL Cloud (+63%) + MSFT Azure (+40%)와 동기화된 가속 사이클이지만, AWS 절대 매출 규모가 1위 ($150B run rate vs Azure ~$95B vs GCP ~$80B).
→ 모니터링 채널: Q2 26 Earnings Release AWS 매출 + 백로그 disclosure (7월 말 예정), AWS re:Invent 2026 (12월) 사전 시그널
→ 뉴스 키워드: "AWS Q2 growth", "AWS backlog Q2", "Anthropic AWS deal", "Trainium customers"

키워드 2: AWS OPM 37.7% sustainability + Trainium 효율 게인 정량화

Q2 AWS OPM 37%+ 유지 시 → Q1 회복 sustainable, 35% 이하 → AI infra 추가 압박. Trainium2 token throughput 4x 개선 효과가 Q2 P&L에 어떻게 반영되는지 추적. Jassy "tens of billions CapEx 절감 + 수백 bps OPM advantage" 가이드의 정량 입증 시점. Trainium 매출 mix 증가 → AWS GPM 변곡점 가능.
→ 모니터링 채널: Q2 26 AWS OPM disclosure, AWS re:Invent Trainium3 양산 발표
→ 뉴스 키워드: "AWS operating margin", "Trainium2 efficiency", "Bedrock workloads", "Trainium3 production"

키워드 3: CapEx Q2 페이스 + FY26 가이드 정량화 시점

Q1 $44.2B → Q2 추정 $48~52B 페이스 → FY26 CapEx ~$195B 도달 가능. Jassy/CFO가 2026 CapEx 정량 가이드 미공개 — Q2 컨콜에서 정량화 시점이 핵심. 비교: MSFT는 CY26 ~$190B 명시, GOOGL은 FY26 $180-190B 명시. AMZN만 정성 → Q2 정량화 시 변동성. CY27 가이드 시기 (예상 Q3 26 컨콜).
→ 모니터링 채널: Q2 26 CapEx 항목 + Jassy/CFO 정량 가이드 멘트, M7 hyperscaler peer 비교
→ 뉴스 키워드: "Amazon CapEx Q2", "AWS data center 2026", "Trainium capacity expansion", "AMZN 2027 guide"

키워드 4: 광고 +22% 가속 + Netflix Audiences 임팩트

Q2 광고 +22% 이상 sustained 시 → Google YouTube 위협 수준. Netflix Audiences 신규 매출 기여 + Rufus 광고 monetization (sponsored prompt 20% 사용자 대화 지속) 확장. TTM $70B+ 광고 매출이 $80B+로 도달 가능 — Stores 마진 다각화 vector.
→ 모니터링 채널: Q2 광고 매출, Amazon Ads upfront 시즌 발표 (5-6월)
→ 뉴스 키워드: "Amazon Advertising Q2", "Amazon Audiences Netflix", "Rufus advertising", "AMZN ad upfront 2026"

키워드 5: LEO 상업 launch (Q3) + 비용 trajectory + Apple 통합

LEO Q2 비용 +$1B (NA seg) → Q3 launch → Q4 cost capitalize 시작. 첫 매출 인식 시점 (Delta 2028 시작 등) = 단기 매출 zero, 장기 옵션. Apple iPhone/Watch satellite service 통합이 가장 큰 catalyst. 250+ 위성 + 20+ 추가 launch 예정. Globalstar 인수 진척 추적.
→ 모니터링 채널: Amazon LEO 위성 추가 발사, 상업 launch 발표, Apple satellite service deal 디테일, Globalstar 인수 close
→ 뉴스 키워드: "Amazon Leo launch", "Amazon Globalstar", "Amazon Apple satellite", "Delta Leo Wi-Fi"

키워드 6: Stores 마진 회복 + Unit growth +15% sustainability + Prime Day 2026

Q2 unit growth +12%+ 유지 시 → covid 이래 사이클 유지, +10% 이하 → 둔화. Q2 NA OPM 8% 도달 + International OPM 4% 도달 시 → 마진 사이클 정상화 입증. Whole Foods + grocery same-day 신규 카테고리 매출 기여. Prime Day 2026 (Q2 대부분 국가) 결과가 Q2 매출 가이드 +16~19% 달성의 핵심 변수.
→ 모니터링 채널: Q2 NA + International OPM disclosure, Same-day perishables 매출, Prime Day 2026 결과
→ 뉴스 키워드: "Amazon Prime Day 2026", "Amazon grocery", "Amazon NA margin Q2", "Whole Foods expansion"

키워드 7: Anthropic 평가 변동성 + IPO 가능성

Q1 Anthropic 평가차익 $16.8B → Q2 추가 평가 변동 시 EPS 변동성 확대. Anthropic 차기 펀딩 라운드 또는 IPO 가능성 시 분기별 변동 큰 폭. 영업 정상화 EPS 추세 추적이 핵심 (일회성 노이즈 분리). AMZN $100B+ deal + GOOGL 지분 = Anthropic 듀얼 hyperscaler 의존 전략의 수익 배분 모델 검증 필요.
→ 모니터링 채널: Anthropic 펀딩 뉴스, Q2 OI&E line item disclosure, Anthropic 매출 공개 시점
→ 뉴스 키워드: "Anthropic valuation", "Anthropic IPO", "Amazon equity gain", "Anthropic Series funding"
