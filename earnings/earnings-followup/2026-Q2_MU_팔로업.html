Micron FY26 Q3 팔로업 리포트 (2026-Q2)

안내: v6 스킬 룰 적용 첫 팔로업 산출물. 1Q26 리뷰(earnings-review/2026-Q1_MU_리뷰.md)의 항목 5(독자 전망), 항목 7(관전포인트), 향후 관찰 포인트의 핵심을 압축 — 풀 디테일은 풀 리뷰 참조.

① 1Q26 (FY26 Q2) 실적 — 비트/미스/보합 요인

(1) 트리플 비트 — 사상 최대급
→ 매출 $23.86B (가이던스 $18.7B 대비 +27.6% / 컨센 $20.07B 대비 +18.9%)
→ Non-GAAP GPM 74.9% (가이던스 ~68% 대비 +6.9pp)
→ Non-GAAP EPS $12.20 (가이던스 $8.42 대비 +45%) — 마이크론 사상 최대 폭 EPS Beat 중 하나
→ OCF $11.9B / Adj. FCF $6.9B (전 분기 +77%) 동반 record

(2) 비트 핵심 동인 — 가격이 전부
→ DRAM ASP +mid-60% QoQ (Bit는 mid-single digits 부합) — ASP가 매출 성장의 약 90% 설명
→ NAND ASP +high-70% QoQ (Bit low-single digits) — DRAM보다 더 폭등
→ 산업 전체 공급 부족 + 우호적 믹스 + 비용 실행 결합 (CFO 인용)
→ 4 BU 모두 record + 모두 +50%+ QoQ. CDBU +139% QoQ로 폭발 1위, MCBU OPM 76% (1년 전 1% → +75pp 점프)

(3) 미스/보합
→ 미스 항목 부재 (모든 KPI 가이던스·컨센 동시 비트)
→ 회사가 의도적으로 침묵한 항목: Q4 이후 GPM forward outlook (이게 폭락 트리거 — 항목 ③ 참조)

② 지난 분기 (4Q25 = FY26 Q1) 핵심 팔로업 사항

→ ⚠️ 첫 작성 — 4Q25 리뷰 자료 없음 (v6 룰 적용 전 분기, 풀 리뷰 미존재)
→ 다음 분기(FY26 Q4) 팔로업부터 본 1Q26 리뷰 자료 기반 추적 정상화

③ 1Q26 핵심 변화 (3개 카테고리)

(1) 산업/사이클
→ 메모리 산업 전체 슈퍼사이클 입증 — 마이크론(3/18) → 삼성 DS OPM 38% (4/7) → SK하이닉스 OPM 71.5% (4/23) 6주간 3사 동시 record 발표
→ HBM3E + HBM4 100% sold out CY2026 (회사 공식) — 가격 가시성 multi-year 확보
→ 첫 5년 SCA 체결 — 메모리 spot 가격 시대 종료 시그널, 산업 구조 변화
→ DRAM/NAND 산업 bit 출하 CY2026 +20%대 (직전 outlook 대비 상향)
→ "Supply-demand tight beyond CY2026" — CFO 명시

(2) 사업부 mix
→ HBM4 12H NVIDIA Vera Rubin mass production 진입 (3/16) — 2일 후 실적 발표
→ CMBU(Cloud/HBM) 비중 32% / CDBU(Core DC) 24% 합산 56%로 데이터센터 매출 압도
→ MCBU(Mobile/Client) GPM 79% / OPM 76% — 4 BU 중 최고 수익성 (모바일 LPDDR5X / Client SSD ASP 폭등)
→ AEBU(Auto/Embedded) 매출 record — 자동차+산업 합산 분기 $2B 초과

(3) Narrative / Multiple
→ 시각 전환 cyclical → secular — 6주간 평균 PT $267 → $517 (+94%)
→ Strong Buy 만장일치 (38명 중 매도 0건) — DA Davidson $1,000 / Melius·Lynx $825 / TD Cowen $660
→ ★ 폭락 1차 트리거 = GPM 피크 시그널 (Q4 이후 forward outlook 회사 침묵) → -30%
→ 회복 결정적 catalyst = SK하이닉스 1Q26 발표(4/23) GPM 79.3% — "산업 전체 GPM 도약" reframe
→ FY26 EPS 컨센 $25 → $53 / FY27 EPS 컨센 $35 → $100 (양쪽 모두 약 +180~200% 상향)

④ 2Q26 (FY26 Q3) 핵심 팔로업 사항

(1) ★ GPM 81% 가이던스 달성 + Q4 forward outlook 톤 (1순위)
→ 이번 폭락의 1차 트리거가 GPM 피크 시그널이었던 점 감안, 단순 81% 달성보다 Q4 GPM 코멘트가 진짜 시그널
→ Bull: "Q4도 80%+ 유지 / FY27 H1 plateau" → secular 10x+ multiple 강화 → 추가 +30% 가능
→ Bear: Q3 81% 미달 (78~80%) + Q4 declining 인정 → 3/30 패턴 재발 (-15~-20%)
→ 모니터링: Micron IR 컨퍼런스콜 transcript (CFO Q&A 톤), 뉴스 키워드 "GPM sustainability" / "Q4 gross margin guidance"

(2) HBM4 12H ramp 진척 + 16H 출하 시점
→ NVIDIA Vera Rubin platform 출하 일정 연동 — Vera Rubin 양산 가속 시 HBM4 ramp 동반 가속
→ 마이크론 HBM 점유율 (현재 ~20%) → 25%+ 도달 가능 여부 (SK하이닉스 70% 점유율 대비 다크호스)
→ HBM4 16H (48GB) 샘플링 → 양산 전환 시점 첫 코멘트 가능성
→ 모니터링: TrendForce HBM 분기 보고서, NVDA FY27 Q1 어닝 (5/20) 후 HBM 코멘트, Bloomberg 뉴스 "HBM4 production ramp"

(3) 추가 SCA 체결 발표 — 어떤 Hyperscaler까지
→ 첫 5년 SCA 후속 체결 가능성 (2~3건 추가 시 multi-year 가시성 본격)
→ 가능성 후보: Microsoft, Google, AWS 중 — 어느 hyperscaler가 마이크론과 SCA 체결하는지가 핵심
→ 모니터링: 마이크론 IR 보도자료, Hyperscaler 분기 컨콜 (4월 말~5월 초 발표 후 SCA 언급 여부), 뉴스 키워드 "Strategic Customer Agreement" / "5-year supply"

(4) FY27 가이던스 첫 시그널
→ FY26 Q3 컨콜에서 FY27 매출/EPS 정성적 톤 — "exceptional growth continues" vs "moderation expected" 어휘 선택 결정적
→ FY27 OP $115B 컨센 미달 시 sell-the-news 가능성
→ FY27 CapEx +$10B 상향이 FY28 공급 과잉 우려 추가 부각 가능성
→ 모니터링: CFO 컨콜 발언, Bloomberg/Yahoo Finance 컨센 즉시 갱신 추적

(5) NAND 공급-수요 갭 정량화 + Singapore fab 진척
→ "Demand significantly in excess of supply" 워딩 변화 (강도 / 약화) 추적
→ Singapore NAND fab 착공 진척 — H2 CY2028 일정 유지 / 슬립 여부
→ Singapore HBM 패키징 가동 시점 — CY2027 의미 있는 기여 일정 변화
→ 모니터링: Trendforce NAND 가격 보고서, 마이크론 IR Singapore fab 업데이트
