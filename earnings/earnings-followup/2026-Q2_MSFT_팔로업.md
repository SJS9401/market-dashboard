> ⚠️ **첫 작성 안내**: 본 종목 직전 분기(FY26 Q2) 리뷰 자료가 워크스페이스에 부재 → **(2) 지난 분기 핵심 팔로업 사항 섹션 자동 생략**. v6 룰 적용 첫 사이클이므로 본 리포트가 baseline. 차기 분기(FY27 Q1 = CY 3Q26) 발표 후 작성 시 본 리포트의 (4) 키워드들이 (2)로 이월되어 사후 검증.

## (1) FY26 Q3 (=CY 1Q26) 실적 — Beat/Miss/In-line 동인 분해

**All-around Beat — CFO Amy "exceeded expectations across revenue, operating income, and EPS"**. 매출 $82.9B (+18% YoY, +15% CC)는 컨센 $81.5B 대비 **+$1.4B Beat**, GAAP EPS $4.27 (+23% YoY)은 cons $4.06 대비 **+$0.21 Beat**, 영업이익 $38.4B (+20%, +16% CC). OPM **46.3%** (+60bps YoY) — 매출 +18% > 비용 +9% (OpEx 기준)으로 **operating leverage 명확 입증**. 5/7 Beat 분포: 매출·Productivity & BP·Intelligent Cloud·Azure·EPS·OpInc 모두 Beat, MPC만 In-line. AI ARR **$37B (+123% YoY)** — Satya 직접 시인.

**진짜 헤드라인은 Cloud + RPO 시그널**. Microsoft Cloud $54.5B (+29% YoY, +25% CC) 분기 매출 비중 66% 도달. **Commercial RPO $627B (+99% YoY)** — Q2 FY26($625B, OpenAI deal 일회성)에서 안정. CFO: "*25% will be recognized in revenue in the next 12 months, up 39% year-over-year*" = 향후 12개월 인식분 **약 $157B 확보** = Cloud 매출 가시성 sector best. **Azure +40% YoY (+39% CC) — 3분기 연속 +39%+ 가속 sustainable**의 1차 검증. Q4 FY26 가이드 +39~40% CC + CFO "second half acceleration" 시사로 추가 가속 가능.

**그러나 Microsoft Cloud GPM은 5분기 연속 하락**: Q3 FY25 69% → Q3 FY26 66% (-300bps 누적), Q4 FY26 가이드 **64% (추가 -200bps)**. CFO 직접 시인 "AI infrastructure investment + GitHub Copilot usage". GitHub Copilot **6월 1일 usage-based pricing 전환** = 단기 GPM 추가 압박. **CapEx CY26 ~$190B 가이던스** (component pricing $25B 포함) — GOOGL FY26 $180-190B와 동일 수준. Q3 FY26 단독 $31.9B (sequential decline due to finance lease timing), Q4 FY26 가이드 >$40B로 재가속. CFO "*remain constrained at least through 2026*" = **AI 인프라 사이클 정점 미도래**.

## (3) FY26 Q3 핵심 변화

**산업·사이클 측면**: AI hyperscaler arms race 정량 확정. **MSFT CY26 CapEx ~$190B = GOOGL FY26 $180-190B 동일 수준** = AI 인프라 사이클 정점 미도래의 정량 시그널. CFO 직접 가이드 "remain constrained at least through 2026" + Sundar 동일 시그널 = **공급 제약이 천장이지 수요 둔화가 아니다**. Azure +40% 3분기 연속 + RPO $627B (+99% YoY) = **2~3년 매출 가시성 sector best**. 동시에 Microsoft Cloud GPM 5분기 연속 하락 = **마진 사이클 압박은 진행 중**, 2027 변곡점 가능성. 소비자 BU(Windows·Xbox·Search)는 **3분기 연속 둔화** — 메모리 가격 + Windows 10 EOL 기저효과.

**사업부 mix 변화**: Intelligent Cloud 매출 비중 **41.8% (vs 38.2% Q3 FY25) — +360bps**. P&BP 비중 42.3% (안정), MPC 비중 15.9% (-340bps, 약화). M365 Copilot **20M+ 좌석 도달 (분기 신규 +250% YoY 사상 최대)** — Accenture 740K (Copilot 사상 최대 단일 win) + Bayer/J&J/Mercedes/Roche 90K+ 각각. **Foundry 다중 모델 전략**: 10K+ 고객 다중 모델 사용, 5K open source 사용, 300+ 고객 1T+ tokens 처리. **Maia 200 + Cobalt CPU 양산 본격화** — Iowa·Arizona DC live, DC 절반 이상 deployed, NVIDIA 의존도 감소 시그널. AI ARR **$37B (+123% YoY)** = OpenAI 매출 + Azure AI services + Copilot 종합.

**Narrative·Multiple 측면**: Satya "agentic computing era" + "platform shift" 톤 = AI 사이클 narrative 강화. 기존 "AI 수익화 미증명" 우려 → **20M Copilot 좌석 + 740K Accenture + AI ARR $37B**로 정량 반박. Foundry **Anthropic + OpenAI 양립** 전략 = OpenAI 의존도 분산 + 고객 선택권 차별화. 셀사이드 **Strong Buy 컨센 진화** (Q2 FY26 Buy → Q3 Strong Buy 트랜지션, 11→14명, +3명) — Wedbush·BofA Strong Buy 진입 + UBS·MoffettNathanson **Hold→Buy 등급 상향**. PT 평균 $510 → $540 (+5.9%). **AI hyperscaler 톱 클래스** narrative 견고화 (GOOGL과 동등 수준).

## (4) Q4 FY26 (=CY 2Q26) 핵심 팔로업 사항 (5~6 키워드)

### 키워드 1: **Azure +39~40% CC 가이드 재가속 + Capacity timing**
3분기 연속 +39~40% CC 가속이 Q4에도 sustainable한지가 가장 핵심 시그널. **Q4 +41%+ 도달 시 → 4분기 연속 가속 + RPO 인식 가속**, +37% 이하 → 가속 사이클 종료 시그널. CFO "second half calendar 2026 acceleration" 명시 → Q4 FY26은 acceleration의 시작 분기. capacity timing(datacenter delivery)이 핵심 변수.
→ 모니터링 채널: Q4 FY26 Earnings Release (7월 말 예정), Microsoft Build (5월), Azure capacity 발표
→ 뉴스 키워드: "Azure growth Q4 FY26", "MSFT capacity datacenter", "OpenAI Azure consumption", "Microsoft Build 2026"

### 키워드 2: **Microsoft Cloud GPM 64% 가이드 검증 + GitHub Copilot 6/1 가격 전환**
CFO 가이드 Q4 64% (Q3 66% → -200bps 추가 하락). 5분기 연속 하락 트렌드가 멈출지가 핵심. **Q4 65%+ 도달 시 → AI 투자 흡수 능력 입증, 변곡점 진입 시그널**, Q4 62% 이하 → FY27 OpMargin 가이드 위협. GitHub Copilot 6/1 usage-based 전환 임팩트가 가이드에 반영됐는지 검증. Maia 200 + Cobalt 양산 확대로 자체 칩 비중 증가 → Cloud GPM 변곡 동인 가능.
→ 모니터링 채널: Q4 FY26 Cloud GPM, GitHub Copilot 매출 disclosure, Microsoft Build Maia/Cobalt 발표
→ 뉴스 키워드: "Microsoft Cloud margin", "GitHub Copilot pricing", "Maia 200 deployment", "Cobalt CPU"

### 키워드 3: **CapEx Q4 FY26 >$40B + CY26 $190B 페이스 + FY27 가이드 정량화**
Q4 FY26 $40B+ 도달 + CY26 누계 페이스 검증. **CY26 $190B의 25% = ~$48B/quarter 페이스 필요** → Q4 FY26 $40B는 baseline 미달, **2H CY26 (=Q1+Q2 FY27) 분기당 ~$60B 페이스 필요**. **FY27 가이드 정량화 시점 = Q1 FY27 컨콜 (2026년 10월 말 예상)** — "double-digit growth" 정성 가이드를 정량으로 변환할 시점. GOOGL FY26 + AWS Q2 등 hyperscaler peer CapEx와의 비교도 핵심.
→ 모니터링 채널: Q4 FY26 CapEx + Q1 FY27 컨콜 (10월 말), GOOGL Q2 26 컨콜, AWS 분기 CapEx
→ 뉴스 키워드: "Microsoft CapEx Q4 FY26", "FY27 CapEx guide", "data center investment 2027", "hyperscaler CapEx"

### 키워드 4: **M365 Copilot 좌석 25M+ 도달 + ARPU 변화**
20M+ 좌석 + Q3 신규 +250% YoY = Q4 25M+ 도달 가능. 큰 고객 50K+ 좌석 4x YoY 진행 + Accenture 740K 같은 mega-deal 추가. M365 Copilot **분기 신규 좌석 페이스 + ARPU 변화 + churn**이 ARR 사이클 검증의 핵심. CFO "ARPU growth led by E5 and M365 Copilot" — Copilot이 단순 add-on이 아닌 ARPU 동인.
→ 모니터링 채널: Satya 분기 발언, Microsoft customer wins 발표, Microsoft Build Copilot 신기능
→ 뉴스 키워드: "M365 Copilot seats Q4", "Copilot enterprise wins", "M365 ARPU"

### 키워드 5: **VRP $900M 정상 흡수 + FY27 추가 헤드카운트 감축**
Q4 FY26 VRP $900M 일회성 비용 (COGS $350M + OpEx $550M). CFO "headcount declining YoY" 가이드 + FY27 OpEx 가이드 mid-to-high single digits = **추가 VRP 발표 가능성**. Q4 실제 VRP 비용이 $900M 가이드 대비 변동 시 + FY27 추가 VRP 시 → operating leverage 강화 시그널.
→ 모니터링 채널: Q4 FY26 VRP 실제 비용, FY27 OpEx 가이드, headcount trajectory
→ 뉴스 키워드: "Microsoft layoff 2026", "Microsoft retirement program", "MSFT headcount decline"

### 키워드 6: **소비자 BU Q4 가이드 verification + Trough 통과 여부**
Q4 가이드: Windows OEM mid-to-high teens decline + Xbox low-teens decline + Search ads ex-TAC high-single-digits. Satya "winning back fans" 톤은 strategic stance. **Q4 결과가 가이드 하단(-15% Windows) 도달 시 → 추가 헤드윈드 sustained**, 가이드보다 좋게 -10% 권역(trough 통과 시그널) 시 → Q1 FY27 base effect 변곡 가능. 메모리 가격 + Windows 10 EOL 기저효과 + Game Pass 가격 변화의 trade-off.
→ 모니터링 채널: Q4 FY26 More Personal Computing 결과, Xbox Game Pass 변화, PC market data (IDC/Canalys)
→ 뉴스 키워드: "Windows OEM Q4 decline", "Xbox Game Pass", "PC market 2026", "Bing 1B MAU"
