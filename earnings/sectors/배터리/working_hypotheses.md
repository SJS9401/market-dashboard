# 배터리 섹터 핵심 워킹 가설

> 본 파일은 배터리 섹터(섹터 T1)의 핵심 워킹 가설을 기록한다. Stage 2 섹터 분석 직후 in-place로 갱신한다.
> 가설은 분석을 진행하는 backbone이며, 매 분기 새 데이터로 검증된다.
> 최종 갱신: 2026-05-04 20:50 KST (LGES·삼성SDI·에코프로비엠·포스코퓨처엠 1Q26 리뷰 풀 사이클 완료, 가설 3 정밀화 — NCA·고객 mix 차별화 한정으로 정밀화)

---

## 가설 1: 미국 ESS 시장은 사실상 한국 셀 독점 구간 진입 — IAA·OBBBA 정책 + 중국 견제로 27.2H까지 공급 부족

- **형성 시점**: 2026-Q1 (LGES + 삼성SDI 동일 시각 + DAOL 컨퍼런스콜 명시)
- **검증 상태**: **강화**
- **핵심 증거**:
  → LGES ESS 매출 +237% YoY / 26년말 50+GWh Capa 가이던스
  → 삼성SDI ESS 매출 +45% YoY / 4Q26부터 미국 LFP ESS 본격 양산
  → 미국 ESS 수요 25년 90 GWh → 30년 160 GWh, 데이터센터 ESS 9 → 40+ GWh (DAOL Q&A)
  → OBBBA 48E ITC: 현지 ESS 50%+ 조달 시 Capex 40% 인센티브 (~'35)
  → IAA: '26.3 의안 발표 → '27.2H 발효 예상 (역내산 강제)
  → DAOL 명시: "미국 시장의 사실상 한국 셀 수주 독점 구간 지속될 것으로 예상"
- **반증 위험**: 
  → IAA 발효 지연 (11월 중간선거 결과 의존)
  → OBBBA 강화 일정 변경 ('27부터 50→55% 강화 예정)
  → 중국 셀의 미국 진입 우회 경로 (멕시코·캐나다 경유 등)
  → ESS 팩 조립 병목 장기화 (LGES 1Q26 ESS 가이던스 +30% vs 실적 +12% 미달)
- **다음 분기 체크 포인트**:
  → 미국 IAA 의안 발표 후속 진척 (3Q26 의회 상정 여부)
  → 11월 미 중간선거 결과 → 정책 영속성 결정
  → 2Q26 LGES ESS 매출 +40% QoQ 도달 여부 (병목 해소 시그널)
  → 삼성SDI 미국 SPE 1라인 가동률 / 4Q26 LFP 양산 진척

## 가설 2: 한국 양대 셀 메이커는 정반대 사이클 — LGES는 사이클 진입(적자전환), 삼성SDI는 사이클 회복(적자축소)

- **형성 시점**: 2026-Q1
- **검증 상태**: **강화** (1Q26 실적으로 명확 입증)
- **핵심 증거**:
  → LGES OPM 추이: 1Q25 +5.6% → 1Q26 -3.2% (적자 진입, 9분기 만의 적자)
  → SDI OPM 추이: 1Q25 -13.7% → 3Q25 -19.4% (저점) → 1Q26 -4.4% (적자 축소 가속)
  → SDI 당기순이익 1Q26 +561억 흑자전환 vs LGES -944억 적자
  → SDI는 ESS·소형·전자재료 흑자 비중 37% (LGES 30% 미만)
  → SDI는 삼성디스플레이 지분법손익 1,146억원으로 안정적 영업외 수익 (LGES는 영업외 약함)
- **반증 위험**:
  → LGES 4Q25 -15.9% → 1Q26 -10.5% 자동차 OPM 회복 시작 — 사이클 차이가 일시적일 가능성
  → 27년부터 수렴 시나리오: LGES 27 OPM 11.5% / SDI 27 OPM 6~9% → LGES가 다시 우위
- **다음 분기 체크 포인트**:
  → 2Q26 양사 OPM 격차 변화 — 컨센 LGES +5%·SDI -2% 시 격차 7%p
  → 4Q26 양사 흑전 시점 일치/차이 (LGES 4Q26 OPM 8.5% vs SDI +2.1%)
  → 27년 본격 흑자 회복 시 LGES vs SDI 우위 재확인

## 가설 3: 양극재 업체는 메탈 가격 상승의 시차 수혜 — 셀 메이커와 사이클 분리 (NCA·고객 mix 차별화 업체 한정)

- **형성 시점**: 2026-Q1 (에코프로비엠 1Q26 OPM 3.5% 회복 vs LGES/SDI 적자)
- **검증 상태**: **부분 검증 (정밀화)** — 1Q26 풀 분석 완료. 에코프로비엠 본업 흑자 vs 포스코퓨처엠 본업 적자 → 가설은 NCA 중심 + 고객 mix 차별화 업체에 한정
- **2026-05-04 (포스코퓨처엠 분석 후 정밀화)**:
  → **에코프로비엠 본업 OPM +3.5%** (NCA 70%+ + 삼성SDI 의존 + 헝가리 양산) → 셀 메이커와 분리 ✓
  → **포스코퓨처엠 양극재 본업 OPM -0.3%** (NCM 90%+ + GM 얼티엄셀 가동중단) → 셀 메이커와 동조 ✗
  → 메탈가 시차 효과(코발트 +140%대)는 양사 모두 받지만, 본업 흑전 여부는 mix·고객 차별화가 결정
- **핵심 증거** (2026-Q1 풀 검증):
  → 메탈 가격 추이 (회사 IR p.9):
    - 코발트: 25.1 23$/kg → 25.11 50$ → 26.1~3 55$ (+139%)
    - 수산화리튬: 25.6 6$/kg → 26.3 11$ (+83%)
    - 니켈: 25 평균 15$ → 26.1 18$ → 26.3 17$ (+13%)
  → **OPM 직접 비교**: 에코프로비엠 +3.5% 흑자 vs LGES -3.2% / SDI -4.4% 적자 — 명확히 분리
  → 에코프로비엠 1Q26 OP 209억 (FnGuide 컨센 98억 대비 **+113% Beat**)
  → 일회성 제외 본업 OPM 추이 (Hana 분석): 1Q25 -4.5% → 4Q25 0.3% → 1Q26 **2.9%** — 9분기 만에 본업 흑전
  → "리튬 가격 상승에 따른 재고자산 평가손실 환입 효과 약 30억" (KB) — 재고 평가충당금 잔여 249억 → 추가 환입 가능
  → 양극재 ASP 메탈가 +10% 이상 시차 반영 (KB 2Q26 추정)
  → 사업부 mix EV 80% → 63% 빠른 분산 (PA 19% / ESS 16%) — EV 의존도 축소도 사이클 분리 강화 요인
- **반증 위험**:
  → 메탈 가격 하락 시 양극재 ASP 동시 하락 → 즉각 마진 압박 (Bear 시나리오 핵심)
  → 셀 메이커가 ASP 협상력 강화 시 양극재 마진 압박 (특히 LGES와의 가격 협상)
  → 미국 EV 보조금 종료 영향 — 동사 미국 노출도 10% 미만이라 제한적
  → 28E P/E 86~127배 vs 글로벌 동종 평균 48배 — 멀티플 부담은 가설과 별개로 잔존
- **다음 분기 체크 포인트**:
  → 2Q26 에코프로비엠 OPM 3% 유지 여부 (14개사 평균 추정 3.5%)
  → 메탈 가격 2Q26 안정 vs 추가 상승 (코발트 55→60$ 추가 상승 시 ASP +15% QoQ 가능)
  → 헝가리 1라인 5월 가동 후 가동률 (2Q말 30%+, 3Q말 50%+ 도달)
  → 한국 양극재 4사 cross-ref (포스코퓨처엠·엘앤에프·코스모신소재 1Q26 발표 후) — 양극재 사이클 분리 일반화 검증
