## 논점 1: AMPC 의존도 vs 본 영업이익 흑전 시점 — 진정한 사이클 회복 시그널은 무엇인가?

- **첫 발생**: 2026-Q1 (LGES + 삼성SDI 1Q26 리뷰 동시 발견)
- **현재 상태**: 미해결 — 2~3분기 추가 데이터 필요
- **Bull 시나리오**:
  → 미국 IAA 발효('27.2H)로 한국 셀 수주 확대 → AMPC 안정 유지
  → 27년부터 AMPC 제외 본 영업이익 흑전 시작 (LGES 27 +518 / SDI 27 +932 셀사이드 평균)
  → 2032년까지 AMPC 단계적 축소 (35→26.3→17.5→8.8→0$/kWh) — 그 사이 본 영업이익이 충분히 회복
  → KB 시나리오: AMPC 2032년 이후 지급 기한 연장 가능
- **Bear 시나리오**:
  → AMPC 제외 본 영업이익 1Q26: LGES -398 / SDI -236 → 사상 최악 수준
  → 11월 미 중간선거 결과에 따른 IRA 정책 변경 리스크
  → KB Bear-case: AMPC 2032년 이전 조기 폐지 가능
  → AMPC 없으면 한국 셀 업체는 27년에도 본 영업이익 BEP 수준에 그칠 가능성
- **트리거 (해결 조건)**:
  → 3Q26~4Q26 AMPC 제외 본 영업이익 흑전 도달 여부 — 도달하면 Bull 시나리오 강화
  → 11월 미 중간선거 결과 + 27.2H IAA 의안 발효 일정 확정
  → 28년 본 영업이익 BEP vs 흑자 폭 (LGES 28F OP +6.5조 / SDI 28F OP +2.6조 컨센)

## 논점 2: 한국 양대 셀 메이커 정반대 사이클 — 구조적 차이인가, 일시적 mix 차이인가?

- **첫 발생**: 2026-Q1
- **현재 상태**: 추가 증거 필요 — 27년 데이터로 검증 가능
- **Bull (구조적 차이) 시나리오**:
  → SDI는 ESS·소형·전자재료 흑자 비중 37% (LGES 30% 미만) — 본질적 mix 차이
  → SDI 삼성디스플레이 지분법손익 안정적 (1Q26 1,146억원) — LGES 영업외 약함
  → SDI 부채비율 79% vs LGES 140% — 우월한 재무구조
  → SDI 1Q26 +40% 컨센 Beat vs LGES -49% 컨센 Miss — 사이클 차별화 명확
- **Bear (일시적 mix 차이) 시나리오**:
  → LGES도 4Q25 -15.9% → 1Q26 -10.5% 자동차 OPM 회복 시작 — 따라잡을 수 있음
  → 27년 셀사이드 OPM: LGES 11.5% / SDI 6.3% → LGES가 다시 우위
  → 28년: LGES 12.7% / SDI 10.1% → 격차 축소
  → STLA JV 청산 + 일회성 보상금이 1Q26 LGES 적자에 큰 비중 → 정상화 시 격차 축소
- **트리거 (해결 조건)**:
  → 27년 본격 흑자 회복 시 LGES vs SDI OPM 격차 (>5%p 시 LGES 우위 회복, 이내 시 격차 유지)
  → 4Q26 양사 분기 흑전 시점 일치/차이 (LGES 4Q26 +8.5% vs SDI +2.1% 셀사이드)
  → 28년 사이클 정점 시 양사 OPM (LGES 14% vs SDI 11~14%)

## 논점 3: 양극재 업체 28E P/E 86~127배 정당성 — 차세대 소재 가시화로 정당화 가능한가?

- **첫 발생**: 2026-Q1 (에코프로비엠 1Q26 풀 분석 후 형성)
- **현재 상태**: **본격 형성** — 시장 반응 1~2주 관찰 후 인뎁스 분석 단계 진입 예정
- **Bull 시나리오** (DS·Kyobo·NH·KB·Hana 등 9개 BUY 그룹):
  → 헝가리 5만톤 capa + 유럽 TCA(27년)·IAA(30년) 정책으로 27~30년 본격 매출 확대
  → 차세대 소재 (전고체 황화물계 27 양산 / LMR 양극재 / 고전압 미드니켈 1Q28 양산) — 멀티플 리레이팅 트리거
  → 28E OP 5,000~6,000억 (KB 5,470 / 27E 2,110 → 28E 5,470, +159%) — 본격 성장 가시화
  → 자율주행·휴머노이드 고에너지 밀도 배터리 수요 → 27E~35E OP CAGR 42% (KB)
  → KB증권 Bull case 290k (P/B 16배), Bear case 170k — 보유 가능 reward/risk
- **Bear 시나리오** (DB·DS(보유)·iM·Trading Buy 그룹):
  → 글로벌 양극재 동종 평균 P/E 48배 vs 동사 86~127배 — 프리미엄 과도
  → ESS 시장 LFP 중심 성장 vs 동사 삼원계 위주 → mix 한계 (iM·Shinyoung)
  → 순차입금비율 106% 돌파 → 헝가리 2공장 증설 시 추가 차입 → 자본 조달 압박
  → 메탈 가격 하락 시 양극재 ASP 즉각 동시 하락 → 마진 압박
  → 차세대 소재 양산 시점 (전고체 27, LMR 미정) 지연 시 멀티플 정당성 훼손
- **트리거 (해결 조건)**:
  → 2Q~3Q26 헝가리 가동률 (2Q말 30%+, 3Q말 50%+ 도달 여부)
  → 4Q26~1Q27 신규 수주 발표 (Mercedes-Benz / CATL 헝가리 / LG에너지솔루션 진입)
  → 27 본격 OP 회복 (2,000억 도달 시 멀티플 정당화 강화)
  → 28F BMW·Mercedes 본격 매출 인식 시점 — 4Q27 vs 1Q28 (셀사이드 분포)
  → 한국 양극재 4사 cross-ref (포스코퓨처엠·엘앤에프·코스모신소재 1Q26 발표 후)
